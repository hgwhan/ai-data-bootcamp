## 4. 패턴 (시각화에서 읽은 것)
- 추이: (계절성 — 봄~여름 증가 등)
        봄인 4월 경에 rentals가 최대치 기록하며, 4월 이전 이후로는 감소하는 모습을 보임
- 관계: (기온과 대여 수의 양의 관계 등)
        기온이 10~15도 일때 대여수가 최대치이며, 기온이 10~15도보다 낮게나 높을 때(춥거나 더울 때) 대여수가 감소하는 모습을 보임
